// v1.0 — Technical Overview

ChaseSwap Protocol

Version 1.0 | 2025 | @chase_swap | t.me/chaseswap

1. Abstract

ChaseSwap is a non-custodial multi-chain DEX on BNB Smart Chain and Polygon. It routes swaps through PancakeSwap and QuickSwap smart contracts directly — no API keys, no intermediaries, no KYC. Flat 0.3% fee, all to liquidity providers.

2. Supported Networks

ChainRouterGas Token
BNB Smart ChainPancakeSwap V2BNB
PolygonQuickSwapMATIC

3. Security

Non-custodial — funds never leave user wallets until signed. Swaps execute through audited PancakeSwap/QuickSwap router contracts. No private keys or personal data stored.

4. Fees

0.3% swap fee goes entirely to the underlying AMM liquidity providers. ChaseSwap charges no additional fees during the launch phase.

5. Disclaimer

Experimental DeFi software. Smart contract interactions carry risk. Not financial advice. Always DYOR.
