Whitepaper

// v1.0 — Technical Protocol Overview

ChaseSwap Protocol

Version 1.0 | 2025 | chaseswap.io | @chase_swap | t.me/chaseswap

1. Abstract

ChaseSwap is a non-custodial, multi-chain DEX aggregator on BNB Smart Chain and Polygon. It routes token swaps through multiple AMM liquidity sources simultaneously, always returning the optimal rate. ChaseSwap charges a flat 0.3% fee with no hidden charges and requires no governance token to use the protocol.

2. Problem

Retail DeFi users face fragmented liquidity across multiple DEXs, chain isolation requiring separate tools for BSC vs Polygon, and opaque routing causing suboptimal rates. Most aggregators solve one of these, rarely all three simultaneously.

3. Solution

ChaseSwap aggregates liquidity via the 1inch Aggregation Protocol API and direct AMM on-chain reads. Our routing algorithm evaluates split routes across up to 5 DEXs per chain and executes the winning path in a single atomic transaction, protecting users from partial-fill risk.

4. Supported Networks

ChainDEX Sources AggregatedGas Token
BNB Smart ChainPancakeSwap V2/V3, BiSwap, BabySwapBNB
PolygonQuickSwap, Uniswap V3, SushiSwap, CurveMATIC

5. Fee Structure

Flat 0.3% on all swaps. Split: 0.25% to AMM liquidity providers; 0.05% to ChaseSwap protocol treasury. No withdrawal fees, deposit fees, or hidden charges of any kind.

6. Price Oracle & Rates

Token prices use CoinGecko public API (free, no key required). Swap output rates use the 1inch Swap API querying on-chain AMM reserves directly. Users see: exact output, price impact, minimum received at their slippage, full route, and estimated gas before confirming.

7. Security Model

Non-custodial — funds never leave user wallets until they sign. Interactions are proxied through audited 1inch AggregationRouterV5 contracts. No private keys, seed phrases, or personal data are ever stored.

  • All transactions user-signed via MetaMask, Trust Wallet, or any Web3 wallet
  • Slippage protection prevents front-running beyond user-defined tolerance
  • Full CertiK security audit planned Q4 2025

8. Governance

Founder-operated during bootstrap phase. Protocol changes announced on X (@chase_swap) and Telegram (t.me/chaseswap) with 7-day community discussion period. Community multisig treasury planned Q1 2026.

9. Revenue & Sustainability

The 0.05% treasury fee is the sole revenue source. At $1M daily volume: $500/day — sufficient to cover all infrastructure costs (GitHub Pages: $0, free API tiers: $0). Treasury held in 2-of-3 multisig.

10. Disclaimer

ChaseSwap is experimental software. Smart contract interactions carry risk. Users are responsible for their own due diligence. Not financial advice. ChaseSwap does not issue any securities or investment products.

// Go from this file to fully live — all free, under 1 hour

STEP 01

GitHub Pages Hosting FREE

  • Create free account at github.com
  • New Repository → name "chaseswap" → Public
  • Upload your index.html file
  • Settings → Pages → Source: main → Save
  • Site is live immediately at:
yourname.github.io/chaseswap
STEP 02

Custom Domain FREE or $10

  • Free: freenom.com → .tk or .ml domain
  • Recommended: namecheap.com → chaseswap.io ~$10/yr
  • Add CNAME DNS record → your GitHub Pages URL
  • In GitHub Pages settings: enter custom domain
  • Enable "Enforce HTTPS" checkbox
STEP 03

1inch API Key FREE

  • Go to portal.1inch.dev → sign up
  • Create App → copy your API key
  • Free tier: 100,000 requests/month
  • Open index.html, find and replace this line:
const INCH_KEY = 'YOUR_1INCH_API_KEY';

Without key: CoinGecko fallback still works!
